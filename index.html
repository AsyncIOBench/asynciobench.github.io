Key Metrics

These are the performance metrics you will see in the benchmark results:

  • IOPS: The number of I/O operations per second, indicating throughput.
  • Latency: The time delay in microseconds for each operation.
  • Bandwidth: The amount of data transferred per second, measured in MiB/s.
